# Website Trust Checklist

Use this checklist to quickly identify why a website loses credibility.

## Message clarity
- [ ] Clear headline explaining what the business does
- [ ] Obvious target audience
- [ ] Location or service area visible
- [ ] Primary call-to-action is clear

## Proof and credibility
- [ ] Real photos (team, projects, office, job sites)
- [ ] Reviews or testimonials
- [ ] Certifications, licenses, or memberships (if relevant)
- [ ] Company name and contact details are easy to find

## Structure and layout
- [ ] Clean visual hierarchy
- [ ] No clutter or competing messages
- [ ] Services are scannable
- [ ] Navigation is simple

## UX basics
- [ ] Mobile layout is readable
- [ ] Buttons look clickable
- [ ] Forms are short
- [ ] No intrusive popups on first load

## Technical trust
- [ ] Fast load time
- [ ] No broken images or layout issues
- [ ] No outdated design elements
